Installation Complete!

MicNoiseGate has been successfully installed on your Mac.

Getting Started:

  1. Open MicNoiseGate from the Applications folder or Launchpad
  2. The app will appear in the menu bar (top-right corner)
  3. Select your microphone input device
  4. Enable noise suppression
  5. In your video conferencing app (Zoom, Discord, Teams), select "MicNoiseGate Mic" as your input device
If "MicNoiseGate Mic" doesn't appear:

The audio system may need to be restarted. You have two options:

Option 1 - Terminal:
sudo launchctl kickstart -kp system/com.apple.audio.coreaudiod

Option 2 - Activity Monitor:
Open Activity Monitor, search for coreaudiod, select it and click the X button to force quit. The system will restart it automatically.

This is only needed once after installation.

First-time setup: If you see a security warning, go to System Settings → Privacy & Security and click "Allow".
